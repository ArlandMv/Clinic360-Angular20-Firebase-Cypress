Acerca de esta Aplicación

Esta aplicación de demostración fue creada para ilustrar la integración de ngx-echarts en una aplicación Angular 20 standalone, así como la generación de PDFs utilizando jspdf.

El panel de control muestra datos simulados de citas clínicas, y esta página demuestra la funcionalidad de exportación a PDF.

Tecnologías Utilizadas:

  • Angular 20 (Standalone Components)
  • ngx-echarts (para visualización de datos)
  • Apache ECharts (motor de gráficos)
  • jsPDF y jspdf-autotable (para generación de PDF)
  • Tailwind CSS (para estilos, simulado con clases básicas)
